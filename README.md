**Основные преимущества приложения ПокерОк на Мак:**

## Вспомогательный Python скрипт pokerok.py

Скрипт `pokerok.py` — небольшая утилита на Python для macOS, которая позволяет проверить и безопасно работать с установочным образом **pokerok.dmg**. Подходит для технического анализа клиента PokerOk на устройствах Apple: MacBook, iMac, Macintosh.

**Возможности:**
* Проверка SHA-256 и размера файла
* Просмотр информации о DMG через `hdiutil`
* Монтирование и просмотр содержимого без открытия Finder
* Копирование `.app` в `/Applications`
* Проверка подписи и Gatekeeper
* Отмонтирование образа

**Установка:**

```bash
curl -O https://raw.githubusercontent.com/PokerOk-GG/Mac/main/pokerok.py
```

| Задача                           | Команда                                         |
| -------------------------------- | ----------------------------------------------- |

| Смонтировать образ и узнать путь | `python3 pokerok.py mount pokerok.dmg`          |
| Просмотреть содержимое образа    | `python3 pokerok.py list pokerok.dmg`           |

| Проверить подпись и Gatekeeper   | `python3 pokerok.py verify pokerok.dmg`         |
| Отмонтировать образ              | `python3 pokerok.py detach -m /Volumes/PokerOk` |
